# 02 — Compliance Checklist (Mevzuat Uyumu Hızlı Tarama)

> Her video script + thumbnail + description + sabitlenmiş yorum bu listeden geçer.

## TR (Türkiye)

### Mevzuat çapaları
- **SB Sağlık Hizmetlerinde Tanıtım Yönetmeliği** R.G. 12.11.2025 (Md. 6 ana yasak listesi)
- **TTB HMEK Md. 11** (hekim reklam yasağı + meslektaşı eleştirme yasağı)
- **Reklam Kurulu** kararları (yanıltıcı reklam, sağlık endişesi sömürüsü)
- **TİTCK** ilaç + takviye gıda yönetmelikleri
- **KVKK** (hasta veri mahremiyeti)
- **Uluslararası Sağlık Turizmi Yönetmeliği** R.G. 26.04.2025 Md. 8

### F1-F16 hızlı kontrol
- [ ] **F1** Sonuç vaadi yok ("Bebek sahibi olacaksınız") — SB Md.6 + TTB Md.11
- [ ] **F2** Belirli klinik/hekim eleştirisi-övgüsü yok — TTB Md.11
- [ ] **F3** Bireysel tıbbi tavsiye yok ("Sizin durumunuzda kesinlikle X") — SB Md.6
- [ ] **F4** Hasta tanıklığı/vaka anlatımı yok — KVKK + SB Md.6 + TTB Md.11
- [ ] **F5** Fiyat/paket/kampanya/indirim yok — SB Md.6 + Reklam Kurulu
- [ ] **F6** Klikbeyt ("Tahmin edemezsiniz") yok — Reklam Kurulu
- [ ] **F7** Korku/panik tetikleme yok ("Yaş ilerliyor, hemen koşun") — SB Md.6
- [ ] **F8** "Mucize / kesin / %100 / garantili" yok — SB Md.6 + Reklam Kurulu
- [ ] **F9** "Tedavi eder / iyileştirir / önler" mutlak iddia (onaysız ürün) yok — SB Md.6 + TİTCK
- [ ] **F10** "En iyi / lider / rakipsiz" üstünlük yok — SB Md.6 + Reklam Kurulu
- [ ] **F11** Belirli ilaç markası/doz/protokol adı yok — SB Md.6 + İlaç Tanıtım Yönetmeliği
- [ ] **F12 (yeni)** "Hocalık / vali-paşa" tonu yok — editöryal (otorite + sıcaklık dengesi)
- [ ] **F13 (yeni)** Görsel klikbeyt yok (kırmızı çapraz, "YAPMAYIN!", şaşkın yüz, oklar) — Reklam Kurulu + thumbnail-rules
- [ ] **F14 (yeni)** İnfluencer/non-hekim isim vererek kınama yok — TTB Md.11 ruhu + mahremiyet
- [ ] **F15 (yeni)** "Ben kullanıyorum/kullanırım" hekim kişisel deneyim yok — TTB Md.11 + SB Md.6
- [ ] **F16 (yeni)** "Hızlı çözüm / 30 günde / tek seansta" pazar dili yok — SB Md.6 + Reklam Kurulu

### Genel kontroller
- [ ] Disclaimer sesli **+** on-screen — "Bu içerik bilgilendirme amaçlıdır; bireysel tıbbi değerlendirmenin yerini almaz."
- [ ] Description'da nötr CTA — tupbebek.com UTM yönlendirmesi, satış dili **YOK**
- [ ] YouTube reklam ifşası — sponsorlu içerik değilse netlik
- [ ] **Donör yumurta TR'de yasak** — bu konu video TR'de geçiyorsa yasal çerçeve net belirtilmeli (Uluslararası Sağlık Turizmi Yönetmeliği Md.8)

## FR (Fransa + francophonie élargie)

### Mevzuat çapaları
- **Code de la santé publique L4113-9** (hekim reklam yasağı)
- **Code de la santé publique R4127** (Conseil de l'Ordre déontologie)
- **Code de la santé publique L1110-4** (secret médical)
- **Code de la santé publique L5122** (publicité médicaments)
- **ARCOM** (eski CSA, 2022 reformu — sağlık iddiası reklam)
- **DGCCRF** (Direction Générale de la Concurrence — yanıltıcı reklam)
- **ANSM** (Agence Nationale de Sécurité du Médicament)
- **RGPD** (veri mahremiyeti)
- **Loi PMA bioéthique 2021** (don d'ovocytes erişim koşulları)
- **FTC** (ABD izleyici için reklam ifşası)
